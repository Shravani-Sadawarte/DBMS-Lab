**to create sequence :**



create sequence emp\_sq start with 1 increment by 1;



create table emp(emp\_id int primary key, name varchar2(10));



***=>in output:*** 

in emp\_id :emp\_sq.nextval



**COUNT MAX SEQUENCE :**

select max(emp\_id) from emp;



**DRAWBACK:** 



{using seq if data of primary table is duplicated it won't show any error }



insert into emp values(\&emp\_id,'\&name');





What if Sequence is Dropped?

=> DROP SEQUENCE EMP\_SQ;





TO add new column PINCODE to the table (emp)

=> alter table emp add pincode int ; 





TRUNCATE :

=> truncate table emp;





RENAME THE TABLE:

=> rename emp to empl;



RENAME ONE PARICULAR COLUM :

=> alter table emp rename column name to emp\_name;
